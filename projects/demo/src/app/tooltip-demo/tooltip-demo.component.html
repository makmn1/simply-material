Plain Tooltips

Plain tooltips provide simple text descriptions for icon buttons.

@for (tooltip of plainTooltips; track tooltip) { }

Rich Tooltips

Rich tooltips can include a subhead, supporting text, and up to two action buttons.

@for (tooltip of richTooltips; track tooltip.supportingText) { }

Programmatic Tooltip Control

Tooltips can be controlled programmatically using the open() and close() methods. When programmatically opened, the tooltip ignores hover and focus events until explicitly closed.

Manual Trigger Mode

This tooltip can only be opened programmatically. Hovering over the button will not trigger the tooltip.

Default Trigger Mode

This tooltip can be opened both on hover and programmatically. When opened programmatically, hover events are ignored until closed.

Disabled Tooltips

Tooltips on disabled elements do not show on hover, focus, or click. However, they can still be opened programmatically using the open() method.
