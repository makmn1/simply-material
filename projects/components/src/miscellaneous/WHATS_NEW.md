# Components

The following is a detailed list of what's new for each component in the Material Design 3 Expressive update.
Note that we only include major additions here. For the full list of changes, review the **Overview** section
of the component's page on the [Material Design site](https://m3.material.io/components).

## App bars
- Search app bar
- Medium flexible and large flexible app bars (replacing medium and large app bars which are being deprecated)
- Updates to the small app bar
- Not in Angular Material

## Buttons

### Button Groups
- New component! Standard and connected button groups
- The connected button group replaces the now deprecated segmented button

### Button
- Added toggle button type
- Added button sizes
- Added square shape
- Added shape morph animations
- Changed default button padding from 24dp to 16dp

### Extended FABs
- New sizes: small, medium, and large

### FAB menu
- New component! Adds options above the FAB when selected

### FABs
- Added medium FAB size

### Icon button
- Added toggle icon button type
- Added button sizes
- Added square shape
- Added shape morph animations
- Added different widths

### Split button
- New component! A button with a menu toggle for additional options

## Loading indicator
- New component! A loading indicator that uses shape morph animations to capture attention

## Progress indicator
- Added wavy shape

## Flexible Navigation Bar
- New component! Replaces the old navigation bar by having a shorter height and can be used in medium window sizes

## Navigation Rail
- Revamped! Now supports collapsed and expanded configurations. Collapsed replaces the original navigation rail. Expanded replaces the navigation drawer

Note: No Material Design component library for the web actually implemented the original Navigation rail / drawer

## Sliders
- Expressive configurations for orientation, shape sizes, and inset

## Toolbars
- Added **docked toolbar** to replace **bottom app bar**
- Added **floating toolbar** with horizontal and floating toolbar
