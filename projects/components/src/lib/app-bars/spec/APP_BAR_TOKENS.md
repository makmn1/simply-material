# App Bar Tokens

Note: The app search bar uses the search component with the default tokens. Refer to SEARCH_TOKENS.md for the search component tokens.

## App Bar Common Tokens

### Color
| Name                                  | Value                                    |
|---------------------------------------|------------------------------------------|
| App bar container color               | `md.sys.color.surface`                   |
| Search container color                | `md.sys.color.surface-container`         |
| Search label color                    | `md.sys.color.on-surface-variant`        |
| App bar container color on scroll     | `md.sys.color.surface-container`         |
| Search container color on scroll      | `md.sys.color.surface-container-highest` |
| App bar container elevation           | `md.sys.elevation.level0`                |
| App bar container elevation on scroll | `md.sys.elevation.level2`                |
| App bar title text                    | `md.sys.color.on-surface`                |
| App bar subtitle text                 | `md.sys.color.on-surface-variant`        |
| App bar leading icon                  | `md.sys.color.on-surface`                |
| App bar trailing icon                 | `md.sys.color.on-surface-variant`        |

### Spacing
| Name                  | Value |
|-----------------------|-------|
| App bar left padding  | `4dp` |
| App bar right padding | `4dp` |
| App bar icon spacing  | `0`   |
| Search left padding   | `8dp` |
| Search right padding  | `8dp` |

### Shape
| Name                    | Value                      |
|-------------------------|----------------------------|
| App bar container shape | `md.sys.shape.corner.none` |

### Size
| Name                | Value  |
|---------------------|--------|
| App bar avatar size | `32dp` |
| App bar icon size   | `24dp` |
