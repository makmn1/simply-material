# Menu Tokens

NOTE: Many of the menu component tokens have been deprecated in favor of using the **list** component for each individual menu item.
The tokens here are simply for the menu container. 

A menu consists of a menu container (tokens shown here), list items, and optional dividers.
For list item and divider tokens, review their respective files (LIST_TOKENS.md, DIVIDER_TOKENS.md).


## Enabled
### Container
| Name                     | Value                             |
|--------------------------|-----------------------------------|
| Menu container color     | `md.sys.color.surface-container`  |
| Menu container elevation | `md.sys.elevation.level2`         |
| Menu container shape     | `md.sys.shape.corner.extra-small` |

## Focused
### Focus Indicator
| Name                           | Value                                       |
|--------------------------------|---------------------------------------------|
| Menu focus indicator color     | `md.sys.color.secondary`                    |
| Menu focus indicator thickness | `md.sys.state.focus-indicator.thickness`    |
| Menu focus indicator offset    | `md.sys.state.focus-indicator.inner-offset` |
