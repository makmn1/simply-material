# Menu

## Overview

Menus display a list of items in a **scannable format** that allows users to make selections quickly.
They should be **easy to open, close, and select** from.
Menus can open from a variety of components, including **icon buttons**, **text fields**, and other interactive elements.

---

## Structure

* **Container:** Holds the list of menu items and optional dividers.
* **List item:** Represents an action or option within the menu.
* **List item leading icon:** Optional icon that appears before the text.
* **List item trailing icon:** Optional icon that appears after the text.
* **List item trailing text:** Optional supporting text that appears at the end of the list item.
* **Divider:** Optional element that separates related groups of items within the menu.

---

## Behavior

**Configurations:**
Menus appear when users interact with a **button**, **action**, or another **control**.
Examples include:

* Button
* Text field
* Icon button
* Selected text

---

## Accessibility

People using assistive technology should be able to:

* Navigate to a menu.
* Open and close the menu.
